# Tools & Formats

Below are just a few tools and formats that we've built over the years.

- [Composite Image builder](https://abcnews.github.io/composite-builder/): combine multiple images in various different ways
- [Odyssey](https://github.com/abcnews/odyssey): a story format to enhance feature articles
- [Base 36 Converter](https://www.abc.net.au/res/sites/news-projects/base-36-props-converter/1.0.0/): convert JSON to base36
